# Materialize NextJS Template

## Start

- start application - `npm run dev` - `yarn run dev`
  >
- start storybook
  - `npm run storybook`
  - `yarn storybook`

---

## Build

- `npm run build`
- `yarn build`

## Directories

- `src/@core`
  - 어플리케이션을 구성하는 코어 components / styles정의 / context / configs / hook / utils
- `src/@fake-db`
  - test용 데이터 파일
